Arduino Gate Opener

Summer 2010, Radius Product Development

Challenge

Radius, a product development firm that I was working for one summer, needed a way for staff members to open parking lot gate for client visits without having to send someone to the parking lot each time. There was a version of arduino controller, but that hasn't been working for months.

Process
Result

Designed ciruitry and electronic packaging; developed web interface and arduino programming; added additional features to enhance user interactions.

Relevant Skills

Arduino, circutry, electronic packaging, soldering, Solidworks, 3D printing.
